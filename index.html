中断是硬件设备中很重要的部分。

对于蓝桥杯的51单片机来说一共有五个中断源。按照优先级来看这五个中断源应该是这样排序的：

外部中断0触发条件是P32按下按键S5 外部中断1触发条件是P33按下按键S4

若想找到一个要打开一个中断从头到尾，需要做打开哪些部分。可以看下图。

同时从上面也能看到各个寄存器，我们一一来说明：

TMOD

该寄存器就是用来选择模式的，见名知意，T开头（定时器）的MOD（模式），就是它是来决定定时中断的模式的见图

IE

IE是啥呢，就是使能中断的嘛！令1就开启。

IP

IP是什么，就是提供一种便利的方法，用来在根据需求要更改优先级的顺序时使用的，在正常优先级的顺序下，先执行高级的，再执行低级的。

TCON

TCON就是中断请求标志寄存器详细如下，给外部中断和

SCON

也是中断标标志寄存器，同时他也有设置串口中断模式的能力（后面这个功能可以和前面提到的TMOD对比学习）

在串口通信中，TI和RI很常用的 其为1时代表发送或接受完成，要手动给它清零，在程序中经常进行编程使用。

谈谈如何在人工智能时代生存

发表于 2024-02-26

​ 近期chatgpt用sora产品再一次震撼了整个科技界，没想到人工智能的迭代进如此迅速，几个月前还是威尔史密斯吃不进拉面的“鬼畜视频”，到现在视频风格在不谈论物理因素的情况下几乎可以说是完美的，在这个迭代速度下，往后的产品我都无法想象。当然随之而来的是，全球的技术人都不可避免的注视着这一技术，并且戏谑的称“程序员的最高境界就是砸掉同行的饭碗”。而对于我一个本科学生来说，我对这件事情的我思考，还是由于我的计组老师，让我们好好想想，有哪些事情是人工智能干不了的，只能由人类去做，我之前也没有深入的思考过，属实惭愧。

1. 深度情感理解与表达

  • 人类情感的深度理解：AI可以通过算法模拟对情感的基本识别和反应，但它们缺乏真正的情感体验和深度理解，无法像人类那样体验情感或对复杂情感做出真正的共鸣。
  • 真诚的同情和安慰：在需要同情和安慰的情况下，人类能提供真诚的情感支持，这是基于人类共通的经历和真实情感体验，AI无法复制这种深层次的人际联系。

2. 道德和伦理判断

  • 复杂的伦理决策：在需要道德和伦理判断的情况下，人类能够考虑到社会规范、文化价值和人情世故等因素，做出复杂的决策。AI虽然可以依据预设的规则作出判断，但它们缺乏对人类社会复杂性的真正理解。
  • 道德责任和后果承担：当决策涉及到道德责任和可能的负面后果时，只有人类能够真正承担责任和后果。

3. 创造性和艺术创作

  • 原创性思考：虽然AI可以模仿和再创造基于已有作品的艺术和音乐，但人类的创造力涉及到从零开始的想象力和原创性思考，这是AI难以达到的。

  • 艺术的情感表达：艺术作品不仅是技术的展示，更是创作者情感和思想的传达。人类艺术家能将个人经历和情感融入作品，与观众建立情感共鸣，这种深层次的情感联系是AI难以实现的。

4. 高度个性化的服务和护理

  • 个性化医疗和护理：在提供个性化医疗和护理服务时，需要考虑患者的个人情况、历史和偏好，以及提供情感上的支持，这些都需要医护人员的人类洞察力和同情心。

  • 个性化教育：教育不仅仅是知识的传递，还涉及到激发学生的兴趣、解决个性化学习问题和情感上的支持。教师能够根据每个学生的特点和需求，提供更加个性化的指导和激励。

这些是在当前大环境下宽泛的认为的AI做不到而需要人类的事物。

从中我们可以套取出计算机领域有哪些事情是人工智能干不了的，只能由人类去做。

1. 创新和创意设计

  • 软件设计与架构决策：虽然AI可以协助生成代码和优化现有算法，但在设计新的软件架构和制定创新解决方案时，需要人类的创造力和前瞻性思考。
  • 用户体验（UX）设计：理解用户需求并创造直观、吸引人的用户界面和体验，需要人类设计师的同理心和创意。

2. 道德和伦理决策

  • 数据隐私和安全：在处理涉及个人隐私和敏感信息的项目时，决定哪些数据应该被收集、如何使用这些数据，需要人类从道德和法律的角度做出考量。
  • 算法偏见与公平性：识别和纠正算法中的偏见，确保技术的公平性和透明度，需要人类的介入和不断审视。

3. 复杂项目管理和协调

  • 跨领域团队协作：管理跨学科团队，调和不同专业背景人员之间的沟通，组织复杂项目的实施，这些都需要人类的社交技巧和领导力。
  • 项目规划和战略决策：制定项目目标、分配资源、评估风险和做出战略性调整，需要人类对业务和技术环境的深刻理解。

4. 研究和教育

  • 科学研究：在新技术、理论的探索和验证过程中，设计实验、解释结果、提出新的研究方向，这些需要研究人员的专业知识和批判性思维。
  • 教育和培训：尽管AI可以辅助教学，提供个性化学习建议，但教师在激发学生兴趣、培养批判性思维、提供指导和反馈方面发挥着不可替代的作用。

5. 高级战略规划和创新

  • 技术前瞻与战略规划：预测技术趋势，制定长期发展战略，以及在竞争激烈的市场中寻找创新机会，需要人类的洞察力和战略思考。
  • 伦理与社会影响评估：评估新技术对社会、文化和环境的潜在影响，制定相应的政策和准则，需要人类的综合判断和道德考量。

总而言之现在的大环境不如从前了，不是之前的学点高级语言，会点数据结构，学点框架，学点库的使用就可以找到本科年薪30个的好工作了，需要针对人工智能无法完成的事情，发掘出自己的价值所在，这样才能站在高峰，保住饭碗！

当然有哪些事情是人工智能干不了的，只能由人类去做这个话题，绝不是一朝就能想明白的，我搜集的信息也一定是不全的，仅供大家寻找思路，也许等到很多年后，才能寻找到答案的真谛。

区块链进度

发表于 2024-02-22 更新于 2024-02-23

Blockchain

研究生学长给我们带来的问题思考的方向。

文章链接：https://pan.baidu.com/s/1wSz9oqnAw6xqBb1sZzA0TQ?pwd=8888

背景大致是什么：

​ 本研究提出的基于区块链的Lamport Merkle数字签名（LMDS）技术，是针对物联网（IoT）医疗设备中敏感数据安全性问题的一个解决方案。随着医疗IoT设备的广泛应用，如何保障这些设备传输的患者数据不被非法窃取或篡改成为了一个亟待解决的问题。LMDS技术通过结合区块链技术的去中心化和不可篡改性质，为医疗IoT设备提供了一种高效且安全的认证机制。

研究意义是什么：

本研究通过引入基于区块链技术的Lamport Merkle数字签名（LMDS）机制，旨在提供一种新的数据安全解决方案，专门针对医疗IoT设备中的敏感数据。这种解决方案的研究意义在于：

  • 它能够利用区块链的去中心化和不可篡改性质，有效防止数据泄露和篡改。
  • 通过Lamport签名和Merkle树的结合使用，提高了认证过程的效率和安全性。
  • 对于医疗行业，这一技术的应用有望提高患者数据处理的安全性和信任度。

怎么改进的：

  1. 技术创新：LMDS技术通过结合Lamport签名和Merkle树，为医疗IoT设备提供了一种更安全、更高效的数据认证机制。这一机制不仅优化了计算过程，降低了计算成本，同时也增强了数据的安全性。
  2. 应用场景的适应性：研究针对医疗IoT设备的特点，对LMDS技术进行了优化和定制，使其更好地适应医疗数据的保护需求。

对比量体现在哪：

对比量主要体现在与传统数据保护技术（如SSL/TLS）和其他区块链技术方案的对比中。通过实验验证，LMDS技术在以下几个方面展示了其优势：

  • 安全性：LMDS技术利用区块链的不可篡改性和Lamport签名的安全性，提供了比传统方案更高级别的数据保护。
  • 效率：在计算开销（CO）和计算时间（CT）方面，LMDS技术通过算法优化，显著减少了必要的计算资源，提高了处理速度。
  • 适用性：特别针对医疗IoT设备的数据特点进行设计，使其在实际应用中更具有可行性。

对于我们好的帮助：

​ 本文提出了一种适用于医疗物联网系统的新型认证技术。建议的解决

方案LMDS旨在通过利用基于区块链的云物联网网络并与患者和医院的医疗保健记

录集成，在没有可信第三方的情况下满足安全需求。该技术促进了云物联网应用部

署、签名生成和签名验证。为了保护物联网网络中的敏感医疗数据，应用了身份验

证技术、LMDSG和LMDSV。该技术具有利用Lamport信号降低CT和Co的优点。该

技术还减少了在每个物联网设备中临时和唯一地生成和存储签名所涉及的苦差事。

此外，使用长哈希值使得恶意用户很难入侵。实验结果表明，对于所提出的技术，

CT，Co在不同的受试者数量下保持稳定。实验分析表明，所提出的LMDS技术降低

了25%的CT和Co，并提高了7%的安全级别。这表明LMDS可以通过记录不同的生

命体征来支持越来越多的受试者，而不会显著增加CT和Co。
